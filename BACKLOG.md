## 📋 עדיפות בינונית (Medium Priority)

### פאזל / סדר (Puzzle / Arrange)
גרירת חלקי תמונה למיקום הנכון — כמו פאזל.
- חלוקת אזורים לחלקים ניתנים לגרירה
- snap-to-position עם אנימציה
- שימוש בתשתית drag קיימת (`input/drag.js`)

### מעקב התקדמות תלמיד (Student Progress Tracking)
שמירת ציונים ומעבר שקופיות בין סשנים.
- localStorage per-student (או ID פשוט)
- כוכבים per-slide (לא רק per-game)
- מסך סיכום עם גרף התקדמות

### ייצוא/ייבוא משחק מלא (Full Game Export/Import)
ייצוא ZIP עם כל התמונות, ההקלטות, וה-JSON.
- אריזת voice blobs + images ל-ZIP
- ייבוא ZIP → שחזור למשחק עובד
- שיתוף בין מורים

### מצב קריאה (Reading Mode)
אזורים מסמנים מילים בטקסט; לחיצה מקריאה את המילה בקול.
- zone-per-word אוטומטי מטקסט
- TTS או הקלטה per-word
- הדגשת מילה נוכחית בזמן הקראה

## 💡 עדיפות נמוכה (Nice to Have)

### דבר או הקלד (Talk or Type)
תלמידים רואים אזור וצריכים להקליד או לומר את התשובה.
- שילוב speech recognition קיים (`audio/speech-recognition.js`)
- שדה הקלדה per-zone
- השוואת טקסט fuzzy לעברית

### וידאו אינטראקטיבי (Interactive Video)
אזורים על פריימים של וידאו — יצירת רגעים אינטראקטיביים.
- הפסקת וידאו בנקודות מוגדרות
- הצגת אזורי zone על הפריים
- המשך וידאו אחרי תשובה נכונה

### שיתוף ופרסום (Sharing & Publishing)
שיתוף משחקים עם מורים אחרים דרך קישור.
- פרסום ל-GitHub Pages (חלקית קיים ב-creator)
- QR code לשיתוף מהיר
- גלריית משחקים ציבורית

### אנליטיקס למורה (Teacher Analytics)
מעקב אחר ביצועי תלמידים בזמן אמת.
- לוח מחוונים (dashboard) למורה
- זמן תגובה, אחוז הצלחה per-zone
- זיהוי אזורים בעייתיים (הרבה שגיאות)

### נגישות מתקדמת (Advanced Accessibility)
- ניווט מקלדת בין אזורים
- תיאור קולי אוטומטי (screen reader)
- מצב ניגודיות גבוהה לאזורים
- תמיכה ב-switch access

### ריבוי שפות (Multi-Language Support)
הרחבה מעברית לערבית, אנגלית, רוסית.
- i18n framework לממשק עריכה
- RTL/LTR דינמי
- TTS engines לשפות נוספות

## 🏗️ שיפורי תשתית (Infrastructure)

### מעבר ל-IndexedDB מלא
localStorage מוגבל ל-5MB — תמונות גדולות יכולות לגרום לבעיות.
- שמירת images ב-IndexedDB (voice store כבר שם)
- מיגרציה שקופה מ-localStorage

### בדיקות E2E
בדיקות אוטומטיות עם Playwright לזרימות משתמש.
- ציור אזור → סימון נכון → שמירה → נגינה
- הקלטת קול (mock) → נגינה
- ייצוא/ייבוא

### PWA / Offline Support
משחקים עובדים ללא אינטרנט.
- Service Worker + cache strategy
- אייקון מסך הבית
- סנכרון כש-online חוזר
