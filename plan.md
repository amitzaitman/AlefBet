AlephBet - מסגרת משחקים חינוכיים לקריאה וכתיבה

Context
מורים בישראל צריכים כלים ליצירת משחקים חינוכיים לילדים בגילאי 3-8 לפיתוח קריאה וכתיבה בעברית. הפתרון: מסגרת קבועה שמספקת API, ו-Claude מייצר קוד משחקים שמשתמש ב-API. כל משחק הוא תיקייה נפרדת.

עקרון מפתח - תשתית צומחת: המסגרת מתחילה מינימלית. כשClaude מייצר משחק שצריך פיצ'ר שלא קיים, הוא מוסיף אותו כמודול לתשתית - כך הוא זמין לשימוש חוזר במשחקים הבאים. כל משחק חדש מעשיר את התשתית.

Module Convention (Rules for Claude When Adding to Framework)
When Claude needs a feature that doesn't exist, it creates a new module following these rules:

One module = one file in the appropriate directory (ui/, render/, input/, audio/, etc.)
Export named functions/classes - no default exports
No dependencies between new modules unless referencing existing core modules
Add the export to index.js so the module is available via the framework bundle
Add corresponding CSS to styles/alephbet.css if the module has visual components
JSDoc comments in Hebrew explaining the module's purpose and API

Creator App - Dynamic System Prompt
The key innovation: the system prompt is built dynamically by scanning the current framework source code.

creator/src/system-prompt.js
js
/**
 * Builds the system prompt by:
 * 1. Scanning framework/src/ for all modules
 * 2. Extracting JSDoc + export signatures from each
 * 3. Composing a prompt that tells Claude:
 *    - What modules exist and their API
 *    - Module conventions for adding new ones
 *    - Game template structure
 *    - Hebrew content guidelines
 */
export function buildSystemPrompt(frameworkModules) {
  return `
אתה עוזר ליצירת משחקים חינוכיים בעברית לילדים.

## המודולים הקיימים בתשתית:
${frameworkModules.map(m => `### ${m.path}\n${m.description}\n${m.exports}`).join('\n')}

## כשאתה צריך פיצ'ר שלא קיים:
1. צור מודול חדש בתיקייה המתאימה ב-framework/src/
2. עקוב אחרי המוסכמות (ייצוא, תיעוד, CSS)
3. הוסף את הייצוא ל-index.js
4. השתמש במודול החדש בקוד המשחק

## מבנה משחק:
כל משחק הוא תיקייה עם: index.html, game.js, game.css (אופציונלי)
game.js מייבא מהתשתית ומכיל את הלוגיקה הייחודית של המשחק.

## כללים:
- כל הטקסטים בעברית
- ממשק RTL
- מטרגט ילדים בגילאי 3-8
- כפתורים גדולים (מינימום 64px) לילדים קטנים
- הודעות מעודדות בעברית
...
  `;
}
In practice: Since the creator runs in the browser and can't scan files directly, we have two options:

Build time: A build script generates a framework-api.json manifest from the source code, which the creator loads
Manual: The developer updates a manifest file when modules are added
I recommend option 1: a script that parses framework source → generates API manifest → creator loads it.
